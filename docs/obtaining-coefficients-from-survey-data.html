Chapter 8 Obtaining coefficients from survey data

8.1 Summary

  • Unweighted estimates may be biased if the inclusion of units in the sample is correlated with the outcome variable conditional on the explanatory variables (Fuller 2009)
  • Weighting by the reciprocals of the unit inclusion probabilities will correct for the bias (Fuller 2009)

8.2 Setup

ConvertFitToResults <- function(fit,dataName,analysisName){
  r <- data.frame(coef(summary(fit)))[-1,-c(3:4),drop=F]
  r[,1] <- sprintf("%s (%s)",
                   formatC(r[,1],digits=3,format="f"),
                   formatC(r[,2],digits=3,format="f"))
  r <- r[,1,drop=F]
  r <- as.data.frame(t(r))
  r$data <- dataName
  r$analysis <- analysisName
  return(r)
}

8.3 Case-control studies

In this case study we will create a dataset popData that has a 1:1 linear relationship between the continuous exposure x and and the continuous outcome y. We also dichotomise y into a binary outcome yBinary.

We then create a dataset casecontrolData that oversamples cases 5x higher than is found in the normal population popData. We will then run both logistic and linear regressions on the original population dataset and the case-control dataset and see how the effect estimates are affected by oversampling cases.

library(data.table)

# Creating population dataset
x <- runif(100000)
popData <- data.table(x)
popData[,y:=x+rnorm(.N)]

# Binary outcome
popData[,yBinary:=0]
popData[y>1,yBinary:=1]
popData[,oversampled:=yBinary]

popData[,inclusionProb := 5]
popData[oversampled==0, inclusionProb:=1]

# Case control dataset
cases <- popData[yBinary==1]
controls <- popData[yBinary==0]

# Cases are sampled 5x higher than normal
casecontrolData <- rbind(cases,cases,cases,cases,cases,controls)
casecontrolData[,id:=1:.N]

# Probability of inclusion

casecontrolData[,f:=1/inclusionProb]
# Full population data set
# Naive linear regression
res <- vector("list",10)
res[[1]] <- ConvertFitToResults(fit <- glm(y~x,data=popData),
                                dataName="Pop",analysisName="Unweighted")
res[[1]]$correlation <- cor(resid(fit),fit$data$oversampled)

# Biased dataset (case control with 5x)
# Naive linear regression
res[[2]] <- ConvertFitToResults(fit <- glm(y~x,data=casecontrolData),
                                dataName="Sample",analysisName="Unweighted")
res[[2]]$correlation <- cor(resid(fit),fit$data$oversampled)

# Biased dataset (case control with 5x)
# linear regression including probability weights
des <- survey::svydesign(id=~id,prob=~inclusionProb,data=casecontrolData)
res[[3]] <- ConvertFitToResults(fit <- survey::svyglm(y~x, design=des),
                                dataName="Sample",analysisName="Weighted")
res[[3]]$correlation <- cor(resid(fit),fit$data$oversampled)

res <- rbindlist(res,fill=T)
setcolorder(res,c("data","analysis","x","correlation"))

knitr::kable(
  res, booktabs = TRUE,
  caption = 'A table of the first 10 rows of the mtcars data.'
)
